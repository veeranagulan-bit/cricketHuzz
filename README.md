🏏 CricketVerse

CricketVerse is a modern and interactive cricket website for cricket fans. It provides information about famous cricket teams, players, World Cup winners, tournaments, and cricket history.

✨ Features

- 🏏 Explore cricket teams
- 🏆 View World Cup winners
- 👤 Explore famous cricket players
- 📖 Learn about cricket history
- 📱 Responsive and user-friendly design
- 🎨 Attractive cricket-themed interface

🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap

🎯 Project Goal

The goal of CricketVerse is to create a simple and engaging platform where cricket fans can explore and learn about the world of cricket.
